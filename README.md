Legend of Link
==============

A fun react experiment.

Uses:

- react
- react-router
- alt flux

Build Chaing:

- Webpack
- Gulp
- BrowserSync

Install & 'Play'
----------------

```
$ npm install
$ npm start
```

Notes for development:

- Components are hot-loaded.
- CSS is auto-injected by browser-sync.
- Edits to Stores/Actions require a hard refresh (because actions get double-bound)
